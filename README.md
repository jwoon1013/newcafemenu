## 2. 연습문제 내용
### 카페 시스템
- 요구사항
    - 고객은 음료 주문할 수 있다.
    - 음료는 아메리카노(1000원), 카페라떼(1500원), 밀크티(2000원), 밀크쉐이크(3000원)가 있다.
    - 직원은 주문 받은 음료를 제조한다.

### 추가
  - 카페의 매출 금액을 알 수 있다.
  - 고객은 주문 시, 금액을 지불해야한다.
  - 고객이 가진 돈 보다 높은 금액의 음료는 주문할 수 없다.
  - 고객이 음료를 마시면 음료는 줄어든다.

출처: 객체지향 사실과 오해(저자: 조영호)
